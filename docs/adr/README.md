# Architecture Decision Records

Registro de las decisiones arquitectónicas significativas del proyecto. Cada ADR documenta el contexto, las alternativas consideradas, la decisión tomada y sus consecuencias.

Los ADRs nunca se editan retroactivamente. Si una decisión cambia, se escribe un nuevo ADR que referencia al anterior con `Supersede: ADR-XXXX`.
